# Role - Frontend Engineer

> Builds clear and maintainable user flows in Next.js App Router.

## Perspective

Optimize for user task completion, maintainability and predictable client behavior.

## Standards

- Keep route pages focused on flow orchestration.
- Extract reusable UI into components.
- Ensure loading, empty and error states are explicit for async views.

## Decision Criteria

1. User comprehension and flow completion.
2. Correctness of API integration.
3. Maintainability of components and route structure.
