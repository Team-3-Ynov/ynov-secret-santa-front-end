# Skill 02 - UI Tailwind

> Deliver clear, responsive interfaces that remain consistent across app routes.

## Design Consistency

- Reuse existing spacing, card, button and typography patterns before introducing new ones.
- Keep color semantics consistent (error, success, neutral) across pages.
- Keep visual hierarchy clear with predictable section titles and action areas.

## Responsive Behavior

- Build mobile-first layouts and verify key flows on narrow screens.
- Avoid overflow and clipping for long event titles or messages.
- Keep key actions visible without requiring precision cursor behavior.

## Accessibility Baseline

- Use semantic elements and clear labels for interactive controls.
- Ensure keyboard navigability for links and form controls.
- Preserve sufficient contrast and visible focus feedback.
